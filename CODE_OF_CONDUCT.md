# Code of Conduct

## Our Pledge

We are committed to creating a welcoming and respectful environment for everyone participating in this project.

## Our Standards

Examples of positive behavior include:

- Being respectful and considerate.
- Accepting constructive feedback.
- Working collaboratively with others.
- Showing kindness and professionalism.

Examples of unacceptable behavior include:

- Harassment or discrimination.
- Offensive or abusive language.
- Personal attacks.
- Any behavior that creates an unwelcoming environment.

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct and may take appropriate action if these guidelines are not followed.
